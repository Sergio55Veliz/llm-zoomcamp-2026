# llm-zoomcamp-2026

## Setup

1. Install Anaconda or Miniconda.
2. Create the Conda environment with Python 3.12:
   ```bash
   conda create -n llm-zoomcamp-2026 python=3.12 -y
   ```
3. Activate the environment:
   ```bash
   conda activate llm-zoomcamp-2026
   ```
4. Install dependencies from `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```

Alternatively, install the package and its dependencies directly:
```bash
pip install .
```

> Note: This repository uses Python 3.12 for the Conda environment. `requirements.txt` holds Python dependencies, and `pip install .` installs the same dependencies via the package configuration.
